HTML na Prática

Ferramentas

Só precisa de um simples editor de texto, como o Notepad no Windows ou outro que vem instalado em qualquer sistema operativo. De um simples editor de texto criar uma página web cheia de cor, efeitos, etc.

Os programs podem ser muito úteis, também, mas já vêm com funcções pré instaladas que poderão alterar aquilo que realmente deseje que apareça nos eu website. No Notepad a responsabilidade é inteiramente sua.


Inspetor de Elementos

A ferramenta de inspecionar elemento é um utilitário que permite visualizar o código-fonte subjacente de qualquer página web. Além disso, você pode usá-la para fazer alterações temporárias e ver os resultados em tempo real, deixando o código-fonte original intacto.

É possível inspecionar elementos pelo celular Android por meio do navegador Chrome. O recurso permite que o usuário visualize os códigos usados no desenvolvimento de uma página da web e copie esses conteúdos para fazer alterações em ambientes externos.


Estrutura Básica

A estrutura principal deve iniciar com a etiqueta "<html>" e terminar com a etiqueta "</html>". O cabeçalho, delimitado pelas etiquetas "<head>" e "</head>", destina-se a conter informações sobre o documento - por exemplo, o título do documento (que é construído com o elemento TITLE e aparece no título da janela do navegador). O corpo do documento, delimitado pelas etiquetas "<body>" e "</body>", destina-se ao conteúdo da página.


Tags

As principais tags do HTML e as mais utilizadas são:

Referente à estrutura

  • <html> - É o elemento raiz que define todo o documento HTML, contém metadados sobre o título e a codificação de caracteres da página.
  • <head> - Faz parte da estrutura básica do documento, sua finalidade é definir o cabeçalho do documento com informações que não serão exibidas dentro do conteúdo da página.
  • <body> - Sua finalidade é definir o corpo do documento, ou seja, tudo que estiver dentro da tag body será mostrado de alguma forma no conteúdo da página.

Referente à formatação

  • <h1>, <h2>, <h3>, <h4>, <h5> <h6> - São TAGS usadas para criar títulos, sendo o <h1> o maior título e o <h6> o menor. Por padrão essas tags vem com formatação em negrito e com fonte maior que o texto normal.
  • <p> - Parágrafo
  • <br> - É usada para criar uma quebra de linha em um parágrafo.
  • <strong> - Coloca o texto em negrito.
  • <i> - É usada para colocar o texto em itálico.
  • <small> - É usado para colocar um texto em fonte reduzida.

Atributos das Tags

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML.

  • <id> - O atributo global id define um identificador exclusivo que deve ser único por todo o documento. Seu objetivo é identificar o elemento ao navegar por âncoras.
  • <blockquote> - O elemento de citação de bloco indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo.
  • <a href=""> - Com o atributo a href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços.

Listas Ordenadas

<ol> - Representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos. Precisa ser acompanhada da tag <li> para listar cada item.


Listas Não Ordenadas

<ul> - Chamamos de lista de marcadores apenas. As tags usadas para criar uma lista não ordenada são <ul> e <li>. A tag <ul> é usada para definir a lista e a tag <li> é usada para cada item da lista.


Links

O Elemento HTML especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.
